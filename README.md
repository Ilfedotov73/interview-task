# interview-task
Тестовое задание на позицию Junior C++ разработчик в Effective Mobile.
Для теста был сгенерирован файл ./tests/inlet.in на 100 записей, который подается в утилиту через перенаправление в поток ввода. Вывод утилиты (для удобства) использует поток вывода, поэтому для вывода результата в .out файл необходимо использовать перенаправление в поток вывода. Реализация списка основана на заголовочном файле 'boost/pool/pool_alloc.hpp' для работы аллокатора, который позволяет избежать теоретически возможной фрагментации памяти, а также ускоряет работу самого списка, т.к. отсутствуют многократные вызовы опера new.

## Сборка
* sudo apt-get install libboost-all-dev -- для pool_alloc.hpp
* g++ -O2 app.cpp linked_list.cpp -o app
* ./app < ./tests/inlet.in > ./tests/outlet.out
